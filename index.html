🔧 Firebase Setup Required

Follow these steps once to connect your cloud database. Takes about 5 minutes.

1
Go to console.firebase.google.com
Sign in with your Google account and click "Add project". Name it e.g. discipleship-tracker. Disable Google Analytics if prompted.
2
Add a Web App
In your new project, click the </> icon. Register the app (any nickname). You'll see a firebaseConfig object — copy it.
3
Enable Firestore
In the left sidebar → Build → Firestore Database → Create database → Start in test mode → choose a region close to Nigeria (e.g. europe-west1).
4
Paste your config below
Replace the placeholder values with your actual config from Step 2.
// Paste your firebaseConfig values here: const FIREBASE_CONFIG = { apiKey: "YOUR_API_KEY", authDomain: "YOUR_PROJECT_ID.firebaseapp.com", projectId: "YOUR_PROJECT_ID", storageBucket: "YOUR_PROJECT_ID.appspot.com", messagingSenderId: "YOUR_SENDER_ID", appId: "YOUR_APP_ID" };
⚠ Before going live, secure your database: In Firebase Console → Firestore → Rules, replace the default rules with the rules shown in the browser console after clicking Save below (press F12 → Console to see them).
